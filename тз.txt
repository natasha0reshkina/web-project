Техническое задание
Задача: создать бота для сервера по информатике.
Функции бота:
-раздача роли по реакции (роль-язык на котором программируешь)
-загрузка картинок(эмблема языка) при обработке сообщений
-отправка личных сообщений с указанием кто может помочь тебе в задаче по твоему языку
-использование api для загрузки картинок (котики там к примеру), переводчик возможно
