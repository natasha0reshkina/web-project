Пояснительная записка
в папке используются файлы:
bot.py-основной код бота
config.py-для хранения разных id, к примеру реакций
keep_alive.py- flask, для хостинга
Procfile- тоже создавала в попытках захостить, в итоге не используется
requirements.txt- загруженные библиотеки
users.csv, users.db - базы данных
тз.txt - техническое задание


пояснение файла bot.py:
-в начале идет функция def translate_text(target, text). Это api переводчик,
по итогу возникли проблемы с его подключением (довольно весомые библиотеки)
-класс User это класс для orm моделей, описывает бд
-класс YLBotClient основной класс бота
-async def on_ready(self) отправляет сообщение о подключении бота (в консоль)
-async def member_join(self, member) приветствует нового участника личным сообщением
-async def on_message(self, message) реагирует сообщение в чате
-async def on_raw_reaction_add(self, payload) реагирует на новую реакцию, выдает роль, добавляет в бд
-async def on_raw_reaction_remove(self, payload) реагирует на удаление реакции, забирает роль, удаляет из бд
